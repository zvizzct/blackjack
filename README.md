#  Blackjack Vite

This is a blackjack game built with Vite and Vue 3.

##  Getting Started

###  Install dependencies

```bash
npm install
```

###  Run the app

```bash
npm run dev
```

### Open the project in your browser

```bash
http://localhost:5173
```

## Production

###  Build the app

```bash
npm run build
```
